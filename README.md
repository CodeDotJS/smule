## Usage

__`It downloads both - audio and video`__

```
 Usage   : smule <source>

 Command :
  -v, --version     :  show version
  -f, --fun         :  no spoiler
  -h, --help        :  for help

 Example :
  $ smule <source>  :  auto detect the filetype and download
```
